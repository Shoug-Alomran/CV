# Shoug Alomran Digital Identity

Premium single-page personal website for Shoug Alomran, built with MkDocs Material for GitHub Pages.

Production domain: https://cv.shoug-tech.com

## Development

```bash
mkdocs serve
```

## Production

```bash
mkdocs build
```

## GitHub Pages

The repository includes a GitHub Actions workflow that deploys the MkDocs build to the `gh-pages` branch. Configure GitHub Pages to serve from the `gh-pages` branch and keep the custom domain set to `cv.shoug-tech.com`.
